# Python - Input/Output
## Description
This project focuses on Python file input/output operations, JSON serialization and deserialization, and data persistence techniques. It covers reading from and writing to files, converting Python objects to and from JSON format, and implementing safe and structured data storage mechanisms.## Files

| File | Description |
|------|-------------|
| `0-read_file.py` | Reads a text file and prints its contents to standard output |
| `1-write_file.py` | Writes a string to a text file |
| `2-append_write.py` | Appends a string to the end of a text file |
| `3-to_json_string.py` | Converts a Python object to a JSON string |
| `4-from_json_string.py` | Converts a JSON string to a Python object |
| `5-save_to_json_file.py` | Saves a Python object to a JSON file |
