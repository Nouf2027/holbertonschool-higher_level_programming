# RESTful API Project

## Project Overview
This project introduces RESTful APIs in a simple way.
It covers how to use APIs, build basic servers, create REST endpoints, and understand basic security and documentation.

## Learning Objectives
- HTTP/HTTPS fundamentals
- API consumption using cURL
- API consumption using Python (requests)
- Building a simple HTTP server
- Building REST APIs with Flask
- Basic API security and authentication
- API standards and documentation
